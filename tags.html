Important html tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough html to be Dangerous. In the process of making it we will learn how to create tables via table and related tags.

The tables below do not list all the HTML tags, but they do list many of the important ones.

Block elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
ol ordered list include an ordered list
li list item include a list element
ul unordered list include an unordered list

Inline elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong bolden text <strong>billion</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com">Learn Enough</a> Learn Enough
img image include image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
